# Remotion Learning Project

## Setup Complete
- Created from `template-helloworld`
- Skills installed in `.claude/skills/`

## Quick Reference

### Commands
- `npm run dev` - Open Remotion Studio
- `npm run build` - Bundle for production

### Key Files to Explore
1. `src/Root.tsx` - Where compositions are registered
2. `src/HelloWorld.tsx` - Main composition with animations
3. `src/HelloWorld/` - Reusable animated components

### Learning Path
1. Run `npm run dev` to open Studio
2. Read through `HelloWorld.tsx` to understand composition structure
3. Modify animations and see live updates
4. Create your own composition

### Skills Available
- `remotion-composition.md` - Creating new compositions
- `remotion-render.md` - Rendering videos
- `remotion-animations.md` - Animation patterns
